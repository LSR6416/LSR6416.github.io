# 200行Html5+CSS3+JS代码实现动态圣诞树

#### 介绍

CSDN开源源码分享动态圣诞树


#### 效果展示：

![输入图片说明](2.gif)

整体圣诞树分为3个部分，书的主干、粒子特效、树上的卡片

树上的卡片(重点)：每一张卡片上都有一个名字，代表圣诞树的叶子，后面可以根据自己需求更改，比如全部改成喜欢人的名字，在JS代码的第五行内更改内容

树的动态旋转通过js实现、主干是html5、样式CSS


#### 使用说明

1.  下载项目源码到本地
2.  点击index.html运行
